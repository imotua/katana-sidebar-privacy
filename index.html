Overview

The Katana MRP Sidebar is a Chrome browser extension that adds a sidebar interface to the Katana MRP web application, providing quick access to inventory items, manufacturing orders, and sales orders without leaving the current page.

This policy explains what data the extension accesses, how it is used, and your rights as a user. We are committed to full transparency — this extension is a simple productivity tool and does not collect, store, or transmit any personal data.

🔑
Data Stored on Your Device

The only data stored by this extension is your Katana MRP API key, which is saved locally in your browser using Chrome's built-in chrome.storage.local API.

This means:

  • The API key never leaves your device except to authenticate directly with Katana's official API
  • It is not accessible to the extension developer or any third party
  • It is stored only on your local machine and is never uploaded to any server
  • You can delete it at any time using the Clear button in the extension popup
📦
Data Accessed from Katana

When you use the extension, it retrieves data from your own Katana MRP account via the official Katana API. This includes inventory levels, manufacturing orders, and sales orders.

This data is fetched solely to display it within the sidebar for your use. It is:

  • Fetched directly from Katana's API using your own credentials
  • Displayed on screen and then discarded — never written to disk or sent elsewhere
  • Not retained between sessions
  • Not accessible to anyone other than you
HTTPS api.katanamrp.com
🚫
Data We Do Not Collect
  • Personal identification information of any kind
  • Browsing history or activity outside of katanamrp.com
  • Analytics, usage statistics, or behavioural data
  • Crash reports or diagnostic telemetry
  • Any data from websites other than katanamrp.com
  • Cookies or tracking identifiers
🔗
Third-Party Services

This extension communicates exclusively with the official Katana MRP API. No other third-party services, analytics platforms, advertising networks, or external servers are used.

The extension does not include any tracking scripts, advertisement SDKs, or external fonts/resources loaded from third-party CDNs.

🗑️
Removing Your Data

You can remove all locally stored data at any time:

  1. Click the Katana MRP Sidebar icon in your Chrome toolbar
  2. Click the Clear button to remove your saved API key

Alternatively, uninstalling the extension from Chrome will automatically remove all associated local storage.
