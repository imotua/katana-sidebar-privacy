Privacy Policy for Katana MRP Sidebar Chrome Extension

Last updated: March 2026

OVERVIEW
The Katana MRP Sidebar ("the Extension") is a Chrome browser extension that adds 
a sidebar interface to the Katana MRP web application. This privacy policy explains 
what data the Extension accesses, how it is used, and your rights as a user.

DATA COLLECTED AND STORED

API Key
The Extension requires the user to provide a Katana MRP API key to authenticate 
requests to the Katana API. This key is stored locally on the user's device using 
Chrome's built-in storage (chrome.storage.local) and is never transmitted to any 
server other than the official Katana MRP API (api.katanamrp.com).

Katana Account Data
The Extension retrieves data from the user's own Katana MRP account including 
inventory items, manufacturing orders, and sales orders. This data is fetched 
directly from Katana's official API and is only displayed within the sidebar 
interface. It is not stored, logged, or transmitted anywhere other than being 
rendered on screen for the user.

DATA NOT COLLECTED
The Extension does not collect:
- Personal identification information
- Browsing history or activity outside of katanamrp.com
- Analytics or usage statistics
- Crash reports or diagnostic data
- Any data from websites other than katanamrp.com

DATA SHARING
The Extension does not share any data with third parties. All communication occurs 
solely between the user's browser and the official Katana MRP API using the user's 
own API credentials.

DATA STORAGE
The only data stored by the Extension is the user's Katana API key, which is kept 
in Chrome's local extension storage on the user's own device. This data does not 
leave the device and is not accessible to the extension developer or any third party.

Users can remove this stored data at any time by:
- Clicking the Extension icon and using the "Clear" button in the settings popup, or
- Uninstalling the Extension from Chrome

THIRD PARTY SERVICES
The Extension communicates exclusively with:
- api.katanamrp.com — the official Katana MRP API, to fetch the user's own 
  business data using their provided API key

No other third-party services, analytics platforms, or external servers are used.

CHILDREN'S PRIVACY
This Extension is intended for business use only and is not directed at children 
under the age of 13.
